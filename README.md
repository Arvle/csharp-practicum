# C# Практикум

Интерактивная платформа для изучения и препода C# с автоматической проверкой кода прямо в браузере.

---

## 👋 Что это такое?

C# Практикум — это веб-платформа, которая позволяет:

**🎓 Студентам:**
- Писать C# код в браузере (редактор как в VS Code)
- Запускать и проверять код одним кликом
- Получать мгновенную обратную связь
- Видеть задания и сроки

**👨‍🏫 Преподавателям:**
- Создавать и управлять заданиями
- Автоматически проверять решения студентов
- Оценивать работы с комментариями
- Отслеживать успеваемость группы

---

## 🚀 Быстрый старт

### Для студентов

1. **Вход** — откройте сайт и введите свой `studentId` (формат: `S1001`, `S1002`, и т.д.)
2. **Выберите задание** — в левой панели показаны все доступные задания
3. **Напишите код** — в редакторе уже есть начальный шаблон
4. **Запустите** — нажмите **Run** или `Ctrl+Enter`
5. **Отправьте на проверку** — нажмите **Submit**, система автоматически проверит результат

### Для преподавателей

1. **Вход** — введите код доступа и выберите группу
2. **Создайте задание** — укажите название, описание, начальный код и ожидаемый результат
3. **Отслеживайте прогресс** — на главной панели видна статистика группы
4. **Проверяйте работы** — нажмите на студента в таблице, чтобы увидеть его решения

## 💡 Возможности

### Редактор кода
- Подсветка синтаксиса C#
- Автодополнение
- Нумерация строк
- Горячие клавиши (`Ctrl+Enter` — запуск кода)

### Автоматическая проверка
- Система запускает ваш код и сверяет результат
- Если результат совпал — задание засчитано
- Если не совпал — видно, что вывела программа
- Преподаватель может проверить и оценить вручную

### Сохранение черновиков
- Ваш код сохраняется автоматически
- При возвращении к заданию код останется на месте
- Черновики хранятся локально в браузере

### Оценивание
- Преподаватель ставит оценку (2-5)
- Можно оставить комментарий к работе
- Студент видит оценку и комментарий

## 📱 Где работает

Платформа работает в любом современном браузере на компьютере, планшете или телефоне. Ничего устанавливать не нужно!

## 🛠 Технологии

Платформа построена на современныхх технологиях:

- **Backend**: Go + PostgreSQL
- **Frontend**: React + TypeScript
- **Редактор**: Monaco Editor (как в VS Code)
- **Выполнение кода**: .NET SDK 8
- **Развёртывание**: Docker
