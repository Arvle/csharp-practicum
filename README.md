# C# Практикум

Интерактивная платформа для изучения языка C# с автоматической проверкой кода.

## Возможности

### Для студентов
- Написание кода прямо в браузере (Monaco Editor)
- Автоматическая компиляция и проверка C# кода
- Мгновенная обратная связь о результате
- Автосохранение черновиков
- Личные проекты

### Для преподавателей
- Управление заданиями
- Статистика успеваемости группы
- Оценивание работ с комментариями
- Просмотр всех решений студентов

## Технологический стек

### Backend
- **Go** - высокопроизводительный бэкенд
- **SQLite/PostgreSQL** - база данных
- **Chi** - роутер
- **.NET SDK** - компиляция C# кода

### Frontend
- **React 18** - пользовательский интерфейс
- **TypeScript** - типизация
- **Vite** - сборка и разработка
- **Monaco Editor** - редактор кода (как в VS Code)

## Запуск

### Требования
- Go 1.21+
- Node.js 18+
- .NET SDK 6.0+ (для компиляции C#)
- pnpm (рекомендуется) или npm

### Установка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/arvle/csharp-practicum.git
cd csharp-practicum

# Запустить бэкенд
cd backend
go mod download
go run cmd/api/main.go

# В новом терминале запустить фронтенд
cd frontend
pnpm install
pnpm run dev
